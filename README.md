## üîç About HeySeen

When you get a PDF, it can be difficult to extract the structured content, especially for technical documents with formulas and images. HeySeen helps by converting these PDFs automatically into editable LaTeX source files. This process includes:

HeySeen relies on technologies like OCR (optical character recognition)‚Äîwhich reads the text from images‚Äîand LaTeX converters to rebuild the document. The tool uses Python and Pytorch frameworks with OCR engines like Tesseract and Surya OCR.
